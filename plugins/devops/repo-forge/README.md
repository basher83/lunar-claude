# repo-forge

Project setup and validation toolkit. Ensures repos are properly configured
with standard tooling across Python, Rust, Terraform, and Ansible projects.

## Skills

| Skill | Status | Purpose |
|-------|--------|---------|
| `mise` | Available | Config patterns, task definitions, language-specific setup |
| `pre-commit` | Planned | Hook configuration and repo setup |
| `gitignore` | Planned | Template generation and validation per project type |
| `git-cliff` | Planned | Changelog config and conventional commit conventions |
| `project-setup` | Planned | Project type identification and orchestrated setup |
