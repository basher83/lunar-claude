# personal-os

Personal workstation operations toolkit for your own machines. Unlike the
infrastructure and homelab plugins, which target remote servers, `personal-os`
holds tooling for the Mac you work *from* — diagnostics, root cause analysis,
and day-to-day operational knowledge that travels across your personal machines.

## Skills

| Skill | Purpose |
|-------|---------|
| `macos-rca` | Structured root cause analysis for macOS failures |

## Installation

```bash
/plugin marketplace add basher83/lunar-claude
/plugin install personal-os@lunar-claude
```
